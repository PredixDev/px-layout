responsive-views
================

A collection of view controls which scale over a wide range of screens (Mobile -> Tablet -> Desktop)
 * SplitView (JQuery / Angular Version)
 * BredcrumbView (JQuery Version)

Dependancies
================
* JQuery
* AngularJS (if you use the AngularJS Wrapper)
