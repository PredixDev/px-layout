Predix UI Responsive Layout Patterns

A layout pattern is pattern that determines how your layout changes at different breakpoints.

How to update your existing layout:

  1. Identify the layout pattern for your app
  2. Use the px-layout component
  3. Select the type you perfer
  4. Replace old markup with new markup
