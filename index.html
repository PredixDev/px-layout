px-responsive-layouts

Includes 3 pre-defined layout wrapper components that enables you to create responsive progressive web apps.

The markup structure of an application can become very messy and deeply nested. So we have identified the essential elements that are needed to create a basic responsive application.

Bundled Elements

  • px-drawer-layout
  • px-header-layout
  • px-layout
  • header - branding / product name
  • section - primary / secondary content
  • aside - primary / secondary sidebar
  • footer - links, copywrite
